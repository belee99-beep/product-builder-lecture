# 저녁 메뉴 추천

이 프로젝트는 간단한 웹 기반 저녁 메뉴 추천 사이트입니다.

## 기능

- **메뉴 추천:** "메뉴 추천" 버튼을 클릭하면 미리 정의된 목록에서 무작위로 저녁 메뉴를 추천합니다.
- **음식 사진 표시:** 추천된 메뉴의 사진을 표시합니다. (현재는 플레이스홀더 이미지 사용)
- **다국어 지원:** 한국어와 영어 간의 언어 전환을 지원합니다.
- **다크/라이트 모드:** 사용자는 다크 모드와 라이트 모드 테마를 전환할 수 있습니다.

## 기술 스택

- HTML
- CSS
- JavaScript (Web Components)

## 계획

1. **`index.html` 수정:** 언어 전환 버튼을 추가합니다.
2. **`main.js` 수정:**
    -   다국어 지원을 위한 번역 및 메뉴 데이터 구조를 추가합니다.
    -   `DinnerRecommender` 컴포넌트를 리팩터링하여 이미지 표시 및 언어 전환 기능을 추가합니다.
    -   언어 전환 버튼에 대한 이벤트 리스너를 추가합니다.
3. **`images` 디렉토리 생성:** 음식 이미지를 저장할 디렉토리를 생성합니다. (이미지 다운로드 실패로 플레이스홀더 사용)
4. **Git에 변경 사항 커밋:** 업데이트된 코드를 버전 제어에 커밋합니다.
5. **원격 저장소에 푸시:** 변경 사항을 원격 저장소에 푸시합니다.

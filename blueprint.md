# 저녁 메뉴 추천

이 프로젝트는 간단한 웹 기반 저녁 메뉴 추천 사이트입니다.

## 기능

- **메뉴 추천:** "메뉴 추천" 버튼을 클릭하면 미리 정의된 목록에서 무작위로 저녁 메뉴를 추천합니다.
- **다크/라이트 모드:** 사용자는 다크 모드와 라이트 모드 테마를 전환할 수 있습니다.

## 기술 스택

- HTML
- CSS
- JavaScript (Web Components)

## 계획

1. **`index.html` 수정:** 페이지 제목과 헤더를 "저녁 메뉴 추천"으로 변경하고, `lotto-generator`를 `dinner-recommender`로 변경합니다.
2. **`main.js` 수정:** `LottoGenerator` 웹 컴포넌트를 `DinnerRecommender`로 교체합니다. 새 컴포넌트는 저녁 메뉴 목록과 추천 버튼을 포함하며, 버튼 클릭 시 무작위로 메뉴를 선택하여 표시합니다.
3. **Git에 변경 사항 커밋:** 업데이트된 코드를 버전 제어에 커밋합니다.
4. **원격 저장소에 푸시:** 변경 사항을 원격 저장소에 푸시합니다.
